# Convert an adjacency matrix to a dagitty string

Convert an adjacency matrix to a dagitty string

## Usage

``` r
dag_matrix_to_dagitty(dag, latents = NULL)
```

## Arguments

- dag:

  Named adjacency matrix (1 for edge, 0 for no edge)

- latents:

  Optional character vector of latent variable names

## Value

A character string in dagitty format
