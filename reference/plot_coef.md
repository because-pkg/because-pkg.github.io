# Plot Path Coefficients

Generic for plotting path coefficients from a fitted model object. For
`because` model objects, see
[`plot_coef.because`](https://because-pkg.github.io/because/reference/plot_coef.because.md),
which produces a caterpillar plot of raw or marginal-effects estimates
with customisable colour schemes and significance highlighting.

## Usage

``` r
plot_coef(object, ...)
```

## Arguments

- object:

  A fitted model object.

- ...:

  Additional arguments passed to the method.
