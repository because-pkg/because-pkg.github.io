# Summarize a Because Counterfactual Object

Summarize a Because Counterfactual Object

## Usage

``` r
# S3 method for class 'because_counterfactual'
summary(object, ...)
```

## Arguments

- object:

  An object of class `because_counterfactual` returned by
  [`do()`](https://because-pkg.github.io/because/reference/do.md).

- ...:

  Additional arguments (ignored).

## Value

A data.frame summarizing the expected value and credible intervals for
each variable.
