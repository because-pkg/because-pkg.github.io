# Latent Child Hook

Modules implement this if a variable is modeled via a latent state (e.g.
occupancy Y -\> z_Y).

## Usage

``` r
is_latent_child_hook(family, variable_name, ...)
```

## Arguments

- family:

  The S3 family list.

- variable_name:

  Name of the variable.

- ...:

  Additional arguments.

## Value

Logical.
